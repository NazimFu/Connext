# **App Name**: MentorConnect

## Core Features:

- User Authentication: Implement login and signup for all user roles.
- Mentor Directory: Display a list of mentors with their expertise and availability. Allow mentees to select a mentor.
- Appointment Scheduling: Mentees can choose preferred meeting date and time and submit a meeting request
- Request Approval: Mentors can approve or decline meeting requests with an optional message to the mentee.
- Verification platform: Our staff will be able to check CV/Essay/MOU verifications uploaded/submitted by the mentees manually and determines if they can be accepted as a valid user of the website.
- Calendar Integration: Generate the meeting in integrated calendar application.
- Link generator: Automatically generate meeting link in selected meeting application.

## Style Guidelines:

- Primary color: A calm blue (#5DADE2) to promote trust and reliability.
- Background color: Light gray (#F0F2F5) to create a clean and professional feel.
- Accent color: A warm orange (#FFB347) to highlight important actions and calls to action.
- Font pairing: 'Space Grotesk' sans-serif for headings, and 'Inter' sans-serif for body text. 'Inter' will create comfortable reading experiences.
- Use clean, modern icons to represent different mentor specialties and meeting types.
- A clean, intuitive layout with clear navigation and a focus on readability.
- Subtle animations for loading states and transitions to improve the user experience.
